# Kinocut post-1.8 program status

**Current release:** Kinocut 1.8.0 is published as of 2026-07-14. The [GitHub
Release](https://github.com/KyaniteLabs/kinocut/releases/tag/v1.8.0), PyPI, npm, and
MCP Registry resolve to 1.8.0. The published surface is 142 MCP tools and 121 CLI
commands.

## Published boundary

The release includes governed AI-video foundations, optional C2PA signing, staged
MCPB foundations, field-safety improvements, the path-based repurpose skill, and the
Hyperframes no-TTY initialization fix. Native MCPB remains staged/local-only rather
than a published, self-contained runtime.

## Remaining program work

The following work is post-release and is not implied to be shipped by the 1.8.0
package:

- Independent Wave-3 verification freeze, audio continuity, and subtitle/graphics QA.
- Asset intelligence, editorial planning, learning reports, and whole-program acceptance.
- `kinocut_sound` S13 host joins, dual-class S14 benchmark evidence, and S15 adversarial acceptance.
- Protected-timeline kernel integration after its named upstream contract and human gate are green.

Use the [roadmap](../../ROADMAP.md) for sequencing and the [parallel execution
plan](../plans/2026-07-12-wishlist-parallel-execution.md) for dependency waves.
