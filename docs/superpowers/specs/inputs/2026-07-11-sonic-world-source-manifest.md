# Sonic World source manifest

**Purpose:** public-safe provenance for the `kinocut_sound` design

**Source date:** 2026-07-11

The source request was a private feature wishlist covering voice synthesis, restoration and
post-processing, script-to-audio assembly, ambience, voice consistency, QA/metadata, and
performance requirements. The public derivative is
`docs/superpowers/specs/2026-07-11-kinocut-sound-sonic-world-design.md`.

The design preserves every capability category in a requirement matrix and adds repository
constraints for local-first execution, explicit provider egress, consent-bound cloning, typed
contracts, deterministic receipts, bounded subprocesses, public-surface parity, and a hard
stop-before-release gate.

The private source file is not tracked and its local path is not published. No hash is claimed for
that untracked input. The public design commit is the reviewable provenance record.
